## 一、如何选择你该看哪份

先按下面这个顺序判断：

1. 你要看的是 `OpenClaw` 还是 `Qclaw`
2. 你要走的是 `云端` 还是 `本地`
3. 如果是本地，再看你用的是 `macOS` 还是 `Windows`

最简单的对应关系如下：

| 你的情况 | 建议阅读文档 |
|------|------|
| 想部署 OpenClaw 到云服务器 | `OpenClaw 云端版` |
| 想在 Mac 本机安装 OpenClaw | `OpenClaw 本地版 Mac` |
| 想在 Windows 上走 WSL2 一键安装 | `OpenClaw Windows 一键脚本版` |
| 想在 Linux 上快速一键安装 | `OpenClaw Linux 一键脚本版` |
| 想在 Windows 本机安装 OpenClaw | `OpenClaw 本地版 Windows` |
| 想横向比较 MiniMax、DeepSeek、Qwen、GLM、Kimi 等模型 | `OpenClaw 模型配置通用说明` |
| 想集中看升级、卸载、常见报错怎么处理 | `OpenClaw 排障与维护通用说明` |
| 想先看 Skills 应该怎么装、先装哪些 | `OpenClaw Skills 安装与推荐通用说明` |
| 想横向比较飞书、QQ、企业微信、Discord 等渠道 | `OpenClaw 聊天渠道总表` |
| 想先按最短路径跑通一遍 | `OpenClaw 新手路线图` |
| 想把 OpenCode、费用、托管版、多 Agent 这些后续内容集中看 | `OpenClaw 进阶玩法通用说明` |
| 想在 Mac 本机安装 Qclaw | `Qclaw Mac 版` |
| 想了解 Qclaw Windows 当前能不能装 | `Qclaw Windows 当前状态版` |
