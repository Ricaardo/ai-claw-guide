## 一、如何选择你该看哪份

先按下面这个顺序判断：

1. 你要看的是 `OpenClaw` 还是 `Qclaw`
2. 你现在处于 `安装前` 还是 `安装后`
3. 如果是安装前，再看你走的是 `云端` 还是 `本地`
4. 如果是本地，再看你用的是 `macOS` 还是 `Windows`

最简单的对应关系如下：

| 你的情况 | 建议阅读文档 |
|------|------|
| 想部署 OpenClaw 到云服务器 | `OpenClaw 云端版` |
| 飞书云 Claw 已经装好，只想看后续怎么配 Skills、定时任务和日常使用 | `飞书云 Claw 安装后使用与配置指南` |
| 想直接复制飞书云 Claw 的定时任务模板 | `飞书云 Claw 定时任务模板大全` |
| 想看飞书云 Claw 常用 Skills 的典型场景和话术 | `飞书云 Claw 常用 Skills 场景手册` |
| 飞书云 Claw 用着用着出问题了，想先快速排查 | `飞书云 Claw 故障排查速查表` |
| 想按自己的用户类型直接抄一套推荐配置 | `飞书云 Claw 推荐配置清单` |
| 想知道飞书云 Claw 每天、每周应该怎么用最顺手 | `飞书云 Claw 每日与每周最佳实践` |
| 想按节奏在 7 天内真正把飞书云 Claw 用起来 | `飞书云 Claw 7 天上手路线` |
| 想先看飞书云 Claw 整套文档该怎么读 | `飞书云 Claw 使用文档总览` |
| 不想看太多命令，只想看最简单版本 | `飞书云 Claw 非技术用户极简版` |
| 想在 Mac 本机安装 OpenClaw | `OpenClaw 本地版 Mac` |
| 想在 Windows 上走 WSL2 一键安装 | `OpenClaw Windows 一键脚本版` |
| 想在 Linux 上快速一键安装 | `OpenClaw Linux 一键脚本版` |
| 想在 Windows 本机安装 OpenClaw | `OpenClaw 本地版 Windows` |
| 想横向比较 MiniMax、DeepSeek、Qwen、GLM、Kimi 等模型 | `OpenClaw 模型配置通用说明` |
| 安装已经完成，但不知道 `openclaw.json`、`USER.md`、渠道、Skills 该怎么继续配置 | `OpenClaw 安装后配置通用说明` |
| 想专门配置 `USER.md`、记忆文件和多 Agent 偏好 | `OpenClaw USER.md 与记忆配置通用说明` |
| 想让 OpenClaw 开机自启、长期常驻 | `OpenClaw 自动启动与长期运行通用说明` |
| 想让手机、平板或远程网络访问，并关注安全 | `OpenClaw 远程访问与安全配置通用说明` |
| 基础 Skills 已经装好，想继续做二次配置 | `OpenClaw Skills 深度配置通用说明` |
| 想集中看升级、卸载、常见报错怎么处理 | `OpenClaw 排障与维护通用说明` |
| 想先看 Skills 应该怎么装、先装哪些 | `OpenClaw Skills 安装与推荐通用说明` |
| 想横向比较飞书、QQ、企业微信、Discord 等渠道 | `OpenClaw 聊天渠道总表` |
| 想先按最短路径跑通一遍 | `OpenClaw 新手路线图` |
| 想把 OpenCode、费用、托管版、多 Agent 这些后续内容集中看 | `OpenClaw 进阶玩法通用说明` |
| 想在 Mac 本机安装 Qclaw | `Qclaw Mac 版` |
| 想了解 Qclaw Windows 当前能不能装 | `Qclaw Windows 当前状态版` |
