# 13. Qclaw 安装状态说明（Windows 版，内测参考）

> **适用对象**：想在 Windows 上使用 Qclaw 的普通用户 | **文档版本**：V1.0
> **更新日期**：2026 年 3 月 11 日 | **文档定位**：Qclaw Windows 当前状态与安装可行性说明

---

## 当前状态

截至 **2026 年 3 月 11 日**，Qclaw 公开内测仅覆盖 macOS。媒体报道虽提到双端计划，但目前没有完整的 Windows 安装教程，官网也没有 Windows 下载入口。不建议安装任何第三方来源的安装包，邀请制内测产品的非官方包存在安全风险。

---

## Windows 用户现在的选择

- **先用 OpenClaw（最推荐）**：走 `WSL2 + Ubuntu + OpenClaw` 路线，当前已有完整官方文档，参考 [05-OpenClaw-本地安装-Windows-零基础指南](../openclaw/install-windows.md)。
- **等官网开放**：关注 Qclaw 官网下载区（`claw.guanjia.qq.com`）是否出现 Windows 下载按钮。
- **如果你同时有 Mac**：先用 macOS 版把产品流程跑通，等 Windows 版开放后再迁移。

---

Windows 版本正在开发中。关注 Qclaw 官网获取最新动态。
