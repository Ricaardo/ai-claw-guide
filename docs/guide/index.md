# 01. OpenClaw 新手路线图

> **适用对象**：第一次接触 OpenClaw，不想一上来被长文档劝退的用户 | **文档版本**：V1.0
> **更新日期**：2026 年 3 月 11 日 | **文档定位**：OpenClaw 新手最短成功路径总览

---

## 先看结论

新手最容易犯的错是一次想装太多。正确顺序只有 5 步：安装 -> 配模型 -> 第一次对话 -> 接渠道 -> 装 Skills。跑通后再看进阶玩法。

---

### 2.1 安装

看对应平台的主文档。

- [06-OpenClaw-模型配置-通用说明.md](../openclaw/model-config.md)

### 2.3 第一次对话

先确认 Web 控制台能打开，OpenClaw 能正常回复。这一步没成功前，不要继续叠渠道、Skills、自动化。

### 2.4 聊天渠道

- [08-OpenClaw-聊天渠道-通用说明.md](../openclaw/channel-guide.md)
