# 04C. 飞书云 Claw 常用 Skills 场景手册

> **适用对象**：已经能在飞书里使用云 Claw，想知道“装了 Skill 之后到底能怎么用”的用户 | **文档版本**：V1.0
> **更新日期**：2026 年 3 月 23 日 | **文档定位**：按使用场景拆解常用 Skills，给出适合直接发到飞书里的话术模板

---

## 先看结论

装 Skill 不是目的，真正有价值的是：

- 你知道这个 Skill 适合干什么
- 你知道在飞书里该怎么说
- 你知道怎么判断它有没有真的被调用

这篇文档就是为这三件事准备的。

---

## 一、最值得先掌握的 7 个 Skills

1. `weather`
2. `browser`
3. `shell`
4. `scheduler`
5. `brave-search`
6. `backup`
7. `skill-vetter`

如果这 7 个你都能顺手用起来，飞书云 Claw 的大部分常见价值已经能发挥出来。

---

## 二、`weather` 怎么用

### 适合做什么

- 查今天天气
- 看未来几天趋势
- 给定时任务提供天气信息

### 适合在飞书里怎么说

```text
今天杭州天气怎么样？
```

```text
帮我看一下这周上海天气变化趋势。
```

```text
请根据今天北京天气，给我一个出门建议。
```

### 怎么判断它真的在工作

如果回答里体现了具体天气信息，而不是泛泛建议，通常说明它被调用了。

---

## 三、`browser` 怎么用

### 适合做什么

- 打开网页
- 查看首页内容
- 提取公告、价格、标题
- 截图或辅助巡检

### 飞书里推荐话术

```text
帮我打开 bilibili.com 看看首页内容，并告诉我你看到的重点信息。
```

```text
帮我访问这个网址，看看有没有最新公告。
```

```text
帮我打开这个网页并截图，然后总结页面上最重要的 3 条信息。
```

### 适合什么场景

- 查看官网公告
- 检查活动页是否更新
- 从网页上提炼信息

---

## 四、`shell` 怎么用

### 适合做什么

- 查看服务器资源
- 查看目录和文件
- 做基础运维检查

### 飞书里推荐话术

```text
帮我看看这台服务器当前 CPU、内存和磁盘使用情况。
```

```text
帮我看看某个目录下有哪些文件。
```

```text
帮我检查某个服务是不是还在运行。
```

### 最适合谁

- 有服务器或云主机
- 想从飞书里直接做轻量巡检

---

## 五、`scheduler` 怎么用

### 适合做什么

- 一次性提醒
- 每天/每周固定提醒
- 组合其他 Skill 做自动任务

### 飞书里推荐话术

```text
请帮我创建一个测试定时任务：10 分钟后提醒我“scheduler 已经生效”。
```

```text
请帮我设置一个定时任务：每天早上 8 点给我发送今天上海的天气预报。
```

```text
请帮我设置一个定时任务：每周五下午 5 点提醒我写周报。
```

### 最容易踩的坑

- 一上来就做太复杂的组合任务
- 还没验证过单个任务就直接上周期任务

---

## 六、`brave-search` 怎么用

### 适合做什么

- 搜索最新资讯
- 查近期行业动态
- 查某个产品最近的更新

### 前提

需要先配置 API Key。

### 飞书里推荐话术

```text
帮我搜索今天关于 OpenClaw 的最新资讯，并用中文总结。
```

```text
帮我搜索最近 24 小时关于 AI Agent 的重要新闻，挑最值得看的 5 条。
```

```text
帮我比较一下最近大家对某个模型的评价重点。
```

### 最适合什么场景

- 行业跟踪
- 每日资讯简报
- 做飞书里的“搜索助手”

---

## 七、`backup` 怎么用

### 适合做什么

- 备份配置
- 备份资料目录
- 准备迁移或升级前保存状态

### 飞书里推荐话术

```text
帮我备份当前 OpenClaw 的关键配置。
```

```text
帮我备份当前配置和资料目录，方便后续恢复。
```

### 为什么重要

很多人都是等配坏了才想到备份，这时已经太晚了。

---

## 八、`skill-vetter` 怎么用

### 适合做什么

- 在安装新 Skill 前先审查风险

### 飞书里推荐话术

```text
我想安装一个新的 Skill，请先帮我检查它是否安全。
```

### 为什么建议第一个装

因为你后面装的所有其他 Skill，理论上都应该先经过它这一层。

---

## 九、推荐的组合场景

### 9.1 搜索助手

组合：

- `brave-search`

飞书里可以说：

```text
帮我搜索今天关于 AI Agent 的重要新闻，并总结成适合发飞书群的版本。
```

### 9.2 日常办公助手

组合：

- `scheduler`
- `weather`
- `browser`

飞书里可以说：

```text
请帮我设置一个定时任务：每天早上 8 点结合今天天气和我待办，生成一份简短早安简报发给我。
```

### 9.3 云主机巡检助手

组合：

- `shell`
- `scheduler`

飞书里可以说：

```text
请帮我设置一个定时任务：每天上午 10 点检查服务器 CPU、内存和磁盘情况，如有异常就通知我。
```

### 9.4 网页巡检助手

组合：

- `browser`
- `scheduler`

飞书里可以说：

```text
请帮我设置一个定时任务：每天下午 3 点检查指定网页是否有公告更新，如有变化就告诉我。
```

## 十、怎么判断一个 Skill 没工作

常见信号：

- 回答明显是泛泛而谈
- 没有体现实时信息
- 没有体现网页内容、系统状态等外部结果
- 明明该定时提醒却完全没执行

这时优先检查：

1. Skill 是否已安装
2. 是否还缺 Key 或授权
3. 你在飞书里的表述是否足够明确
4. 网关是否需要重启

```text

```

