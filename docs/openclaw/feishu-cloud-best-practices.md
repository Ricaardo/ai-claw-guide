# 04F. 飞书云 Claw 每日与每周最佳实践

> **适用对象**：飞书云 Claw 已经能正常使用，想把它真正纳入每天工作流的用户 | **文档版本**：V1.0
> **更新日期**：2026 年 3 月 23 日 | **文档定位**：把飞书云 Claw 从“能用”推进到“每天真的会用”的实践手册

---

## 先看结论

飞书云 Claw 真正发挥价值，不靠你偶尔问一句，而靠你把它放进固定节奏。

最有效的方式不是追求复杂，而是建立这 3 个习惯：

1. 每天固定让它生成或提醒 1 到 2 次
2. 每周固定让它做一次汇总
3. 把高频重复动作逐步交给它

这篇文档不是讲怎么安装，而是讲怎么让它变成你日常的一部分。

---

## 一、推荐的每日使用节奏

### 1.1 早上：让它做开工助手

每天早上最适合用飞书云 Claw 做的，不是大任务，而是“进入状态”。

推荐动作：

- 看天气
- 看今日待办
- 看一份简短简报

推荐一句：

```text
请帮我生成今天的开工简报，结合天气和我今天最重要的 3 件事。
```

如果你已经准备了 `~/.openclaw/data/todo.md`，效果会更稳定。

### 1.2 上午：让它做信息助手

上午适合用来：

- 搜索行业信息
- 看公告
- 查网页变化

例如：

```text
帮我搜索今天关于 AI Agent 的重要动态，并总结最值得关注的 5 条。
```

### 1.3 下午：让它做执行辅助

下午更适合让它帮你：

- 检查网页
- 看服务器状态
- 做小范围资料整理

例如：

```text
帮我检查一下这台服务器的 CPU、内存和磁盘情况。
```

### 1.4 晚上：让它做收尾提醒

晚上不需要复杂功能，一个固定的收尾提醒就很有价值。

例如：

```text
请提醒我整理今天完成了什么、还有哪些没做完、明天第一件事是什么。
```

## 二、推荐的每周使用节奏

### 2.1 周一：启动本周节奏

建议在周一早上做：

- 本周重点目标提醒
- 周会准备
- 风险项整理

推荐一句：

```text
请提醒我整理本周重点目标、关键会议和当前风险项。
```

### 2.2 周中：检查是否偏离

周三或周四最适合做一次中途检查。

例如：

```text
请提醒我检查一下本周进度，看看是否有重要事项正在延期。
```

### 2.3 周五：固定做周报和收尾

这是飞书云 Claw 最适合固定发挥价值的时点。

推荐动作：

- 周报提醒
- 本周完成项梳理
- 下周待办准备

推荐一句：

```text
请提醒我整理本周完成事项、问题和下周计划，按周报结构输出。
```

## 三、不同用户的最佳实践重点

### 3.1 个人用户

重点做：

- 天气提醒
- 待办提醒
- 晚间复盘

不建议一开始就搞复杂巡检。

### 3.2 办公用户

重点做：

- 今日重点提醒
- 日报/周报提醒
- 会议前提醒
- 待办简报

### 3.3 运营型用户

重点做：

- 行业资讯简报
- 竞品网页巡检
- 公告变化提醒

### 3.4 技术用户

重点做：

- 服务器健康检查
- 网页巡检
- 配置备份提醒

---

## 四、最值得养成的 5 个长期习惯

### 4.1 让它固定在同一时间出现

不要让它“想起来才用”。固定时间触发，最容易形成习惯。

### 4.2 让它优先做重复动作

什么最适合交给它：

- 每天都要提醒的
- 每周都要汇总的
- 每次都差不多的话术

### 4.3 先让它提醒，再让它组合执行

顺序应该是：

1. 提醒
2. 查信息
3. 读文件
4. 组合工作流

### 4.4 保持 `todo.md` 和模板文件更新

如果你的 `data/` 目录长期不更新，很多自动化会越来越空。

### 4.5 每周至少复盘一次“哪些任务值得自动化”

一个很实用的问题是：

> 我这周有没有反复做 3 次以上的同类动作？

如果有，基本就值得交给飞书云 Claw。

---

## 五、推荐的日常最小组合

如果你只想维持一套最精简但长期有用的实践，建议保留这 4 个：

1. 每天 8 点天气或开工提醒
2. 每个工作日 9 点今日重点提醒
3. 每周五 5 点周报提醒
4. 每天晚上 6 点收尾提醒

这套已经足够让飞书云 Claw 进入你的日常节奏。

---

## 六、什么情况下说明你已经开始“用起来了”

出现下面任意 3 条，基本就说明不是“装着玩”，而是开始融入工作流了：

- 你每天至少会收到 1 次有效提醒
- 你每周至少有 1 个固定任务依赖它
- 你开始往 `data/` 目录放长期资料
- 你在飞书里已经有固定的使用话术
- 你会主动让它做搜索、网页或巡检类任务

---

## 七、推荐你本周就开始做的版本

本周先完成这 5 件事：

1. 做一个早上提醒
2. 做一个周报提醒
3. 做一个搜索类任务
4. 建好 `todo.md`
5. 建好最小版 `USER.md`

这套最容易落地，也最容易坚持。
