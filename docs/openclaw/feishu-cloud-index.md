> **更新日期**：2026 年 3 月 23 日 | **文档定位**：飞书云 Claw 文档总入口，按“先做什么、出问题看什么、你属于哪类用户”来组织

## 先看结论

如果你已经完成飞书云 Claw 安装，这组文档建议按下面顺序看：

1. 先看使用主线
2. 再看定时任务和 Skills 场景
3. 出问题就看速查表
4. 不知道怎么选配置，就看推荐清单

你不需要一次把所有文档看完，只要找到当前最需要的那一篇。

## 八、一个最短建议

如果你现在只想立刻开始，请直接做这 3 件事：

1. 装 `scheduler`、`weather`、`brave-search`
2. 建一个每天早上 8 点的天气提醒
3. 建一个每周五下午 5 点的周报提醒

做到这里，你的飞书云 Claw 就已经开始产生稳定价值了。
